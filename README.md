# Meu Segundo Cérebro

Tags: [[Segundo Cérebro]] [[Produtividade]] [[Organização]]

### Descrição
---
```C
"Seu cérebro foi feito para ter idéias, não para guardá-las"
Não conheço o Autor
```

A principal idéia é criar aqui meu **segundo cérebro**, de forma que eu possa organizar meus pensamentos e ideias do dia à dia e também relacionados aos meus projetos. Neste momento os pensamentos serão organizados da seguinte forma:
- **Inbox:**  Para tudo que eu capturar rapidamente;
- **Notas:** Para notas processadas;
- **Ideias:** Para insights e pensamentos;
- **Referências:** Para resumos de livros e artigos;
- **Tags:** Para auxiliar na interconexão de ideias, utilizando um sistema de tags (temas chave);
Experiências novas como assistir novos vídeos, ler novos l [[Segundo Cérebro]] [[Produtividivros e o próprio uso da ferramenta podem ajudar na melhor organização do meu segundo cérebro.
### Rotina de uso
---
Para que este sistema funcione, vou seguir a seguinte rotina de uso:
- De manhã criar uma nova nota na pasta **Inbox** com o título da data de hoje. Jogar aqui tudo que eu anotar no meu celular durante o dia;
- Tirar um dia da semana para revisar aquilo que realmente vale a pena ser guardado e excluir aquilo que não vale a pena. Também é interessante revisar ideias antigas para lembrar como resolvi problemas antigos;
Vale a pena tentar outras rotinas também, para aprender novos métodos e criar o seu próprio.
