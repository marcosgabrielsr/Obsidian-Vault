## Sistema de download de arquivos do google drive com a API do Google Drive v3

tags: [[Projeto Py-Drive]] [[Python]] [[Programação]]
### Descrição
---
A ideia aqui é implementar um sistema de download de arquivos com a API do Google Drive v3. Para isso eu vou utilizar a função `download_files` que será responsável por fazer o acesso aos comandos da API. A ideia dessa função é poder baixar os arquivos (o arquivo estando na lixeira ou não) em um path definido e se possível descompactar o arquivo baixado caso ele seja uma pasta, talvez seria até interessante adicionar um sistema de progresso de download.
